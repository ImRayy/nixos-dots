## Applications

- [Hyprland](https://hyprland.org/) or [Niri](https://github.com/niri-wm/niri) - compositor/window-manager
- [Noctalia Shell](https://noctalia.dev/) - top bar, Launcher, Notificatons etc.
- [Foot](https://codeberg.org/dnkl/foot) - terminal
- [Yazi](https://github.com/sxyazi/yazi) - TUI file explorer, [Nautilus](https://apps.gnome.org/en-GB/Nautilus/) - GUI file explorer
- [LazyVim](http://www.lazyvim.org/), [Neovide](https://github.com/neovide/neovide) - TUI & GUI text editor

**Other Apps**: These are available, but I don't actively use. Can be enabled

- [Kitty](https://github.com/kovidgoyal/kitty), [Ghostty](https://ghostty.org/) - GPU accelerated terminal for Wayland/X11

## Installation

**Rebuild System & Home Manager**

```bash
cd nixos-dots/
sudo nixos-rebuild switch --flake .#everfrost
```
